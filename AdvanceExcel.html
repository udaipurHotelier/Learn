Advanced Excel

Data Handling & Cleaning

Remove Duplicates : Go to Data → Remove Duplicates. Select columns, confirm.

Data Validation:Go to Data → Data Validation. Set rules.

Conditional Formatting:Go to Home → Conditional Formatting. Apply colors/icons.

Sort & Filter:Go to Data → Sort & Filter. Arrange/filter data.

Text to Columns:Go to Data → Text to Columns. Split data.

Flash Fill:Go to Data → Flash Fill. Auto-fill patterns.

Pivot Tables & Charts

Pivot Tables :Pivot Tables – Insert → PivotTable. Summarize data.

Pivot Charts:Pivot Charts – Insert → PivotChart. Visualize PivotTables.

Slicers & Timelines:Go to Home → Conditional Formatting. Apply colors/icons.

Grouping Data:Go to Data → Sort & Filter. Arrange/filter data.

Data Analysis & Business Intelligence

Goal Seek:Goal Seek – Data → What-If Analysis → Goal Seek. Adjust inputs.

Scenario Manager:Scenario Manager – Data → What-If Analysis → Scenario Manager. Compare cases.

Data Tables:Data Tables – Data → What-If Analysis → Data Table. Analyze inputs.

Solver :Solver – Data → Solver (enable Add-in). Optimize solutions.

Charts & Visualization

Charts:Charts – Insert → Charts. Choose Bar, Line, Pie, etc.

Dynamic Charts:Dynamic Charts – Use Named Ranges & Tables. Auto-update.

Trendlines:Trendlines – Chart Elements → Trendline. Predict trends.

Dual-Axis Charts:Dual-Axis Charts – Combo Chart option. Compare data.

Power Query & Power Pivot

Power Query : Power Query – Data → Get & Transform Data. Import, clean data.

Merge & Append Queries : Merge & Append Queries – Power Query Editor. Combine data.

Power Pivot : Power Pivot – Power Pivot Tab (enable Add-in). Handle big data.

Relationships : Relationships – Power Pivot → Manage → Relationships. Connect datasets.

Macros & Automation

Record Macros : Record Macros – View → Macros → Record Macro. Automate tasks.

Run Macros : Run Macros – View → Macros → View Macros. Execute macros.

Edit Macros : Edit Macros (VBA) – Alt + F11. Modify code.

Assign Macros : Assign Macros – Right-click button → Assign Macro.

Security & Protection

Protect Sheets : Protect Sheets – Review → Protect Sheet/Workbook. Restrict editing.

Hide Formulas: Hide Formulas – Format Cells → Protection. Secure data.

Restrict Editing : Restrict Editing – Review → Allow Edit Ranges. Control access.
